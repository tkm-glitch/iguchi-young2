📌 GitHub への貼り付け方

  1. 1 こちらをクリックしてGitHubの編集画面を開く
  2. 2 ファイル内の /* ▼▼▼ ここに貼り付け↓ ▼▼▼ */ という行を探す
  3. 3 その次の行の先頭にカーソルを置き、コピーしたコードを貼り付ける
  4. 4 画面下の「Commit changes」ボタンを押せば完了！
